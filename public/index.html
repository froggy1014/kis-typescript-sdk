🏦 Korea Investment Securities API

TypeScript SDK with OpenAPI Documentation

📋 프로젝트 소개

한국투자증권 API를 위한 TypeScript SDK입니다. OpenAPI 3.0 문서와 함께 제공됩니다.

주요 기능

  • 완전한 TypeScript 타입 지원
  • OpenAPI 3.0 자동 문서화
  • Swagger UI 인터페이스
  • 주식 거래 API 통합

🚀 API 서버 접속

이 프로젝트는 백엔드 API 서버입니다. 실제 API는 다음 플랫폼에서 실행되어야 합니다:

Vercel에서 실행 Railway에서 실행 Render에서 실행

📖 설치 및 사용법

NPM 설치

npm install kis-typescript-sdk

기본 사용법

import { KisApi } from 'kis-typescript-sdk';

const api = new KisApi({
  appKey: 'your-app-key',
  appSecret: 'your-app-secret',
  baseUrl: 'https://openapi.koreainvestment.com:9443'
});

// 주식 현재가 조회
const price = await api.getStockPrice('005930');

⚠️ 배포 안내

현재 Netlify에 배포되어 있지만, 이 프로젝트는 정적 사이트가 아닌 Node.js 백엔드 API 서버입니다.

실제 API 기능을 사용하려면 다음 중 하나의 플랫폼에 배포해야 합니다:

  • Vercel - Node.js 런타임 지원
  • Railway - 간단한 배포
  • Render - 무료 티어 제공
  • Heroku - 전통적인 PaaS
